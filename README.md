# rest-express

Приложение для REST API с использованием Express и TypeScript.

## Установка зависимостей

```bash
npm install
```

## Запуск приложения

### Режим разработки

Для запуска сервера в режиме разработки:

```bash
npm run dev
```

Для запуска клиентской части в режиме разработки:

```bash
npm run dev:client
```

### Сборка проекта

Для сборки проекта выполните команду:

```bash
npm run build
```

### Запуск в продакшене

После сборки запустите приложение в продакшене:

```bash
npm start
```

### Дополнительные команды

- Проверка типов: `npm run check`
- Обновление базы данных: `npm run db:push`
